# MyPlaces
Приложение разработанное для добавления своих любимых мест и заведений.
Позволяет находить места на карте, а так же прокладывать до них маршруты. 
 
 ---
 Использовал в проекте:

* MVC
* Frameworks (MapKit, Realm, Cosmos) 
* UITableView
* MapKit View
* UISearch
* UIPickerView
* UISegmentedControl
* AutoLayout
* CocoaPods, SPM
* GCD
